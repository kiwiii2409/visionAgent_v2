# Firefox Web Browser

## Searching the Web

1. Focus the address bar: click it at the top of the window, or press **Ctrl+L**
2. Type your search query
3. Press **Enter** to search with the default search engine (usually Google)

## Reading Search Results

- Google result pages show a main search result area with links, snippets, and sometimes info cards
- Info cards (like BTC price) show the value directly — read the number from the screenshot
- If the page is still loading, wait for results to appear before extracting data
- Scroll the page if needed result not shown, research if needed information not shown in the first three pages
## Important Notes

- Do NOT use `launch_application_tool` to open Firefox if it is already running — switch to it via the dock or Alt+Tab instead
- After typing a search query and pressing Enter, wait 3-5 seconds for results to load
